# Tech Stack: RPG-Web-Term (Island Survival Edition)

## Backend (Серверна частина)
- **Node.js:** Потужне середовище виконання, ідеальне для реального часу.
- **Fastify:** Легкий та надзвичайно швидкий фреймворк, що мінімізує споживання пам'яті (важливо для Render Free Tier).
- **Socket.io:** Бібліотека для двостороннього зв'язку GM та гравців у реальному часі.

## Frontend (Клієнтська частина)
- **Vite + React (TypeScript):** Швидка збірка та сучасний підхід до розробки інтерфейсу.
- **Tailwind CSS:** Для стилізації "безлюдного острова" з мінімальним CSS-кодом.
- **Xterm.js:** (Опціонально) Для створення справжнього термінального досвіду в браузері.

## Storage (Збереження даних)
- **JSON-based Storage (lowdb):** Оскільки ми прагнемо мінімізації ресурсів, збереження стану сесії у JSON-файлах на сервері (або в пам'яті з періодичним збереженням) є чудовим варіантом для MVP.
- **Render Disk (опціонально):** Для персистентності даних на безкоштовному тарифі.

## Deployment & Hosting (Хостинг та Деплой)
- **Render:** Безкоштовний тариф для хостингу веб-сервісу.
- **GitHub Actions:** Для автоматичного CI/CD (тестування та деплой).

## Rationale (Обґрунтування)
- **Ефективність:** Fastify та Socket.io споживають мало пам'яті, що дозволяє уникнути проблем із лімітами Render.
- **Швидкість розробки:** React + Tailwind дозволяють швидко створити стильний та адаптивний інтерфейс.
- **Простота:** Відсутність важкої БД (PostgreSQL/MongoDB) на старті MVP спрощує налаштування та підтримку.
